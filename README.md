# Harkon Byggerådgivning

Website for Harkon Byggerådgivning — professionel byggerådgivning i Nykøbing Falster.

**Stack:** Vite + React 19 + Tailwind CSS v4 + Framer Motion

**Deployed on:** [Vercel](https://harkon.vercel.app)

## Local development

```bash
npm install
npm run dev
```

## Contact form (Formspree)

The contact form sends email notifications via [Formspree](https://formspree.io). No backend required — the form POSTs directly to the Formspree endpoint.

## Deployment

Deploys automatically to Vercel on push to `main`.
